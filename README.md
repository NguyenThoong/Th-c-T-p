# 🚀 Automation Testing Framework - Automation Exercise

Dự án Báo cáo Thực tập Kiểm thử Tự động tại **Viện Khoa học Tính toán và Trí tuệ Nhân tạo (COSARI)**.

## 📌 Công nghệ sử dụng
- **Language:** Java 11+
- **Automation Tool:** Selenium WebDriver
- **Testing Framework:** TestNG
- **Data-Driven Model:** JSON & Google Gson Library
- **Design Pattern:** Page Object Model (POM)
- **Reporting:** ExtentReports / TestNG Report

## 🛠️ Cấu trúc dự án
- `src/main/java/com.vlu.pages`: Chứa các Class Page Object Model
- `src/test/java/com.vlu.helpers`: Lớp tiện ích đọc dữ liệu JsonHelper
- `src/test/java/com.vlu.tests`: 5 Test Modules quản lý 26 Test Cases
- `src/test/resources/TestData.json`: Bộ 26 dữ liệu kiểm thử ngoại vi
